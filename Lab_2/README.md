# Виртуальный датчик для контроля процесса обжига

## Описание проекта

Целью проекта является разработка модели машинного обучения (виртуального датчика), позволяющей прогнозировать концентрацию продукта обжига на основе технологических телеметрических данных. Решение охватывает полный цикл анализа данных: от предобработки и синхронизации измерений до построения и оценки качества прогнозных моделей.

## Используемые данные

В работе использовались следующие наборы данных:

- `data_train.csv` — телеметрические данные технологического процесса (минутная дискретизация)
- `target_train.csv` — результаты лабораторных измерений концентрации (нерегулярные наблюдения)
- `data_test_small.csv` — тестовый набор телеметрии
- `target_test_small.csv` — тестовые лабораторные измерения

## Основные этапы работы

### 1. Разведочный анализ данных (EDA)

На данном этапе были выполнены:

- анализ распределений телеметрических параметров
- исследование поведения целевой переменной
- учет временной задержки лабораторных измерений относительно технологических параметров
- проверка данных на пропуски и выбросы

### 2. Feature Engineering

Для повышения качества моделей были сформированы дополнительные признаки:

- лаговые признаки (временные задержки параметров)
- скользящие статистики (mean, std, min, max)
- производные признаки (diff)
- агрегированные временные характеристики процесса

### 3. Построение моделей

Были обучены и сравнены несколько моделей регрессии:

- **OLS (Linear Regression)**
- **Random Forest**
- **Gradient Boosting**

### 4. Оценка качества моделей

Для комплексной оценки качества прогнозирования использовались следующие метрики:

- **MAE** (Mean Absolute Error)
- **RMSE** (Root Mean Squared Error)
- **MAPE** (Mean Absolute Percentage Error)
- **WAPE** (Weighted Absolute Percentage Error)
- **Directional Accuracy** (точность предсказания направления изменений)

Дополнительно были выполнены:

- анализ остатков моделей
- анализ важности признаков
- визуальный анализ прогнозов и фактических значений

## Результаты

По результатам сравнительного анализа наилучшее качество показала модель **Gradient Boosting**.

Преимущества данной модели:

- наименьшее значение ошибки прогнозирования среди рассмотренных моделей
- хорошее воспроизведение динамики технологического процесса
- приемлемое время обучения и инференса
